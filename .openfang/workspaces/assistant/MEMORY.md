# Long-Term Memory

## Durable User Context
- The user prefers direct execution over long theoretical planning.
- The user likes a sharp, witty, non-corporate assistant voice rather than a soft generic helper.
- The user expects backups before risky or user-facing prompt/workspace edits.
- The user does not want `cargo fmt` run unless they explicitly ask for it.

## Important Reference Facts
- Assistant workspace files live under `~/.openfang/workspaces/assistant/`.
- Template defaults come from the repository, but the live assistant behavior is strongly shaped by the local workspace markdown files.
