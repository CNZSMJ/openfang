## 选股逻辑 - 三因子模型

```
综合得分 = 价值因子×40% + 动量因子×30% + 质量因子×30%
```

### 1️⃣ 价值因子 (40%)

|-----|---------|
| PE | <10: +30分, <20: +20分, <30: +10分, >50: -20分 |
| 股息率 | >5%: +20分, >3%: +10分, >1%: +5分 |

### 2️⃣ 动量因子 (30%)
| 指标 | 加分规则 |
|-----|---------|
| 20日涨幅 | -10%~+20%: +20分, -20%~+30%: +10分, >+30%: -10分 |
| 60日趋势 | 上涨: +15分, 下跌>20%: -15
