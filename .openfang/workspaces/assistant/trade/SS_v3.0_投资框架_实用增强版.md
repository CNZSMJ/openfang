## 二、模块详解

### 1. 初筛清单（过滤明显坑）

```
┌────────────────────────────────────────┐
│  排除标的 (满足任一则排除)               │
├────────────────────────────────────────┤
│  □ PE(TTM) > 100 倍（成长股）或 >30倍（周期股）│
│  □ 流动性 < 日均成交 5000 万            │
│  □ 上市未满 6 个月（新股风险）           │
│  □ 业绩预亏或连续 2 年亏损               │
│  □ 被监管点名/立案调查/ST               │
│  □ 大股东质押 > 80%                      │
│  □ 行业处于下行周期（需宏观判断）         │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│  纳入标的 (满足任一可纳入)               │
├────────────────────────────────────────┤
│  □ 符合当下主线热点（政策/业绩/题材）    │
│  □ 机构持股比例 > 10%（有机构背书）      │
│  □ 近期有业绩预告/订单/并购等催化       │
│  □ 板块热度 5 日涨幅 > 0%（资金关注）   │
└────────────────────────────────────────┘
```

### 2. 三层评分体系（v3.0 透明化）

```
┌─────────────────────────────────────────────────────────────────────────┐
│  价值层 (40/100)  —— 衡量"这家公司好不好"                              │
├─────────────────────────────────────────────────────────────────────────┤
│  基本面质量      [0-100]  权重 33%                                    │
│    = ROE(20%) + 营收增速(20%) + 毛利率(20%) + 现金流(20%) + 行业地位(20%)│
│    · ROE > 20% = 100分, 10-20% = 70分, 5-10% = 50分, <5% = 30分       │
│    · 营收增速 > 30% = 100分, 15-30% = 70分, 0-15% = 50分, <0% = 20分   │
│    · 毛利率 > 40% = 100分, 20-40% = 70分, 10-20% = 50分, <10% = 30分  │
│    · 现金流/营收 > 10% = 100分, 0-10% = 70分, 负 = 30分               │
│    · 行业TOP3 = 100分, TOP10 = 70分, 其他 = 50分                       │
│                                                                         │
│  行业位势        [0-100]  权重 33%                                    │
│    = 行业景气度(40%) + 竞争格局(30%) + 政策支持(30%)                   │
│    · 行业景气度：上升周期100分，平稳70分，下行30分                      │
│    · 竞争格局：寡头垄断100分，寡头70分，充分竞争50分，散户30分         │
│    · 政策支持：大力支持100分，一般70分，限制30分                        │
│                                                                         │
│  估值赔率        [0-100]  权重 33%                                    │
│    = PE百分位(40%) + PB百分位(30%) + 相对大盘(30%)                     │
│    · PE百分位 < 30% = 100分, 30-50% = 70分, 50-70% = 50分, >70% = 30分│
│    · PB百分位 < 30% = 100分, 30-50% = 70分, 50-70% = 50分, >70% = 30分│
│    · 相对大盘超额 > 20% = 100分, 0-20% = 70分, -20-0% = 50分, <-20% = 30分│
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  交易层 (42/100)  —— 衡量"现在买能不能赚钱"                             │
├─────────────────────────────────────────────────────────────────────────┤
│  资金结构        [0-100]  权重 33%                                    │
│    = 主力净流入(40%) + 股东户数变化(30%) + 机构持仓(30%)               │
│    · 主力净流入（5日）> 5亿 = 100分, 1-5亿 = 70分, 0-1亿 = 50分, 负 = 20分│
│    · 股东户数连续下降 = 100分, 持平 = 70分, 上升 = 30分                  │
│    · 机构持仓 > 30% = 100分, 15-30% = 70分, 5-15% = 50分, <5% = 30分    │
│                                                                         │
│  裸K形态         [0-100]  权重 33%                                    │
│    = 趋势强度(30%) + 均线排列(30%) + 关键位置(40%)                      │
│    · 趋势强度：上升趋势100分，震荡70分，下降30分                         │
│    · 均线排列：多头排列(MA5>MA10>MA20) = 100分，纠缠 = 50分，空头 = 30分│
│    · 关键位置：突破压力位100分，接近压力位70分，接近支撑位50分，跌破支撑 = 30分│
│                                                                         │
│  量价配合        [0-100]  权重 33%                                    │
│    = 成交量配合(50%) + 量价背离检测(50%)                                │
│    · 成交量配合：价涨量升 = 100分，价涨量跌 = 50分，价跌量升 = 50分，价跌量缩 = 100分│
│    · 无量价背离 = 100分，有背离 = 30分                                  │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  风险层 (54/100)  —— 衡量"可能亏多少"                                   │
├─────────────────────────────────────────────────────────────────────────┤
│  波动风险        [0-100]  权重 33%                                    │
│    = 20日年化波动率(50%) + 最大回撤(50%)                                │
│    · 波动率 < 20% = 100分, 20-40% = 70分, 40-60% = 50分, >60% = 30分   │
│    · 最大回撤 < 5% = 100分, 5-10% = 70分, 10-20% = 50分, >20% = 30分   │
│                                                                         │
│  事件风险        [0-100]  权重 33%                                    │
│    = 业绩确定性(40%) + 限售股解禁(30%) + 减持计划(30%)                  │
│    · 业绩确定性：已披露预增 = 100分，预亏 = 30分，未披露 = 50分          │
│    · 1年内无限售股解禁 = 100分，3个月内解禁 = 30分，已解禁 = 70分       │
│    · 无减持计划 = 100分，有减持 = 30分                                  │
│                                                                         │
│  流动性风险      [0-100]  权重 33%                                    │
│    = 日均成交额(50%) + 买卖盘深度(50%)                                  │
│    · 成交额 > 10亿 = 100分, 5-10亿 = 70分, 1-5亿 = 50分, <1亿 = 30分   │
│    · 买卖盘差距 < 5% = 100分, 5-15% = 70分, >15% = 30分                 │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3. 档位修正规则（v3.0 细化）

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         档位计算规则                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Step 1: 计算三层平均分                                                  │
│    价值层 = (基本面 + 行业位势 + 估值) / 3                              │
│    交易层 = (资金结构 + 形态 + 量价配合) / 3                            │
│    风险层 = (波动风险 + 事件风险 + 流动性) / 3                          │
│                                                                         │
│  Step 2: 基础档位                                                       │
│    三层均 >= 65 分  →  A级 (强烈推荐)                                   │
│    三层均 >= 55 分  →  B级 (推荐)                                       │
│    任一层 >= 50 分  →  C级 (观察)                                       │
│    任一层 < 50 分  →  D级 (不推荐)                                      │
│                                                                         │
│  Step 3: 强制降级 (一票否决)                                             │
│    □ 价值层 < 40 分  →  降至 D 级                                       │
│    □ 交易层 < 40 分  →  降至 D 级                                       │
│    □ 风险层 < 40 分  →  降至 D 级                                       │
│    □ 估值百分位 > 90% →  强制降至 C 级以下                              │
│    □ 资金连续流出 > 5 日 →  强制降至 C 级以下                           │
│    □ 形态破位 (跌破关键支撑位) →  强制降至 D 级                          │
│                                                                         │
│  Step 4: 特殊升级 (需满足全部条件)                                       │
│    □ 三层均 >= 70 分  →  可升一级                                       │
│    □ 有重大利好催化 (订单/并购/政策)  →  可升一级                        │
│    □ 缩量回踩不破支撑 + 资金开始回流  →  可升一级                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4. 信号输出（v3.0 新增核心）

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          操作信号定义                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  BUY (买入)      条件：                                                  │
│    · 档位 A 级或 B 级                                                  │
│    · 交易层 >= 60 分 (形态配合)                                         │
│    · 价格处于关键支撑位附近 (±3%) 或 突破压力位后回踩                    │
│    · 成交量开始放大 (量价配合)                                          │
│    · 无重大利空消息                                                     │
│                                                                         │
│  WATCH (观察)     条件：                                                │
│    · 档位 C 级                                                         │
│    · 价值层或交易层偏低，但无明显利空                                     │
│    · 等待形态修复或催化出现                                             │
│                                                                         │
│  SELL (卖出)      条件：                                                │
│    · 档位降至 D 级                                                     │
│    · 价格跌破失效位 (止损)                                              │
│    · 达到预设止盈位                                                     │
│    · 出现重大利空 (业绩暴雷/政策打压)                                   │
│    · 资金持续流出 > 10 日                                               │
│                                                                         │
│  HOLD (持有)      条件：                                                │
│    · 已持仓标的                                                         │
│    · 档位未发生强制降级                                                 │
│    · 未触及止盈止损线                                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 四、与 v2.3 框架对比

| 模块 | v2.3 | v3.0 增强 |
|------|------|----------|
| 初筛 | 无 | 新增排除/纳入清单 |
| 评分 | 黑箱 (33/55/33) | 透明化 (给出具体计算公式) |
| 档位 | 简单规则 | 4步修正 (基础+降级+升级) |
| 信号 | 只有 WATCH | BUY/SELL/HOLD/WATCH 明确 |
| 仓位 | 无 | 新增仓位矩阵 |
| 检验 | 无 | 下单前必检清单 |
| 风控 | 失效条件 | 预设止盈止损+自动降级 |
| 跟踪 | 无 | 每日/周/月复盘机制 |

---

## 五、框架局限性（诚实面对）

1. **无法预测黑天鹅** → 预留仓位应对极端行情
2. **历史数据依赖** → 需要结合实时消息面
3. **主观判断仍存在** → 形态识别、均线排列等需要人工复核
4. **需要数据源** → 需要可靠的财务数据、资金流向数据
5. **不是圣杯** → 需要持续复盘优化

---

## 六、下一步建议

1. **获取数据源**：财务数据(PE/ROE/现金流)、资金流向、形态数据
2. **建立评分表**：用 Excel 或 Notion 搭建评分跟踪表
3. **先模拟后实盘**：用历史数据回测 3-6 个月
4. **从小仓位开始**：验证有效后再放大

---

> **核心理念**：SS v3.0 = "先给可执行动作，再给可验证逻辑；用评分闸门和仓位管理保证稳健性"
